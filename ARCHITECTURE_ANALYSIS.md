### 2. ❌ Отсутствуют циркадные ритмы

**Проблема:** Нет 24-часового цикла, нет мелатонина/кортизола.

**Что нужно:**
```rust
struct CircadianClock {
    time_of_day: f64,        // 0.0-24.0 часов
    melatonin: f64,          // Высокий ночью
    cortisol: f64,           // Высокий утром
    sleep_pressure: f64,     // Накапливается за день
}

impl CircadianClock {
    fn update(&mut self, elapsed: f64) {
        self.time_of_day = (self.time_of_day + elapsed / 3600.0) % 24.0;
        
        // Мелатонин: пик в 2-4 часа ночи
        self.melatonin = ((self.time_of_day - 3.0) * PI / 12.0).cos() * 0.5 + 0.5;
        
        // Кортизол: пик в 6-8 часов утра
        self.cortisol = ((self.time_of_day - 7.0) * PI / 12.0).cos() * -0.5 + 0.5;
        
        // Давление сна: накапливается за день
        if self.time_of_day > 6.0 && self.time_of_day < 22.0 {
            self.sleep_pressure += elapsed * 0.01;
        } else {
            self.sleep_pressure *= 0.95; // Снижается во время сна
        }
    }
}
```

**Последствия отсутствия:**
- Питомец может спать в любое время без логики
- Нет естественного ритма бодрствования/сна
- Невозможно моделировать джетлаг или нарушения сна

---

### 3. ❌ Отсутствуют фазы сна (NREM/REM)

**Проблема:** Есть только DeepSleep и LightSleep, но нет циклов и replay.

**Что нужно:**
```rust
enum SleepPhase {
    NREM1,  // Дремота (5-10 мин)
    NREM2,  // Легкий сон (20 мин)
    NREM3,  // Глубокий сон (30 мин) - REPLAY ПАМЯТИ
    REM,    // Быстрый сон (10-20 мин) - СНОВИДЕНИЯ
}

struct SleepCycle {
    phase: SleepPhase,
    phase_duration: f64,
    cycle_count: u32,  // 4-6 циклов за ночь
}

impl SleepCycle {
    fn update(&mut self, memory: &mut MemoryModule) {
        match self.phase {
            SleepPhase::NREM3 => {
                // REPLAY: воспроизвести паттерны дня
                for memory in memory.episodic.iter() {
                    self.replay_spike_pattern(&memory.spike_pattern);
                    // Усилить синапсы, участвовавшие в этом паттерне
                }
            }
            SleepPhase::REM => {
                // СНОВИДЕНИЯ: случайный шум → генерация странных паттернов
                let random_spikes = self.generate_random_spikes();
                let dream = self.interpret_spikes(random_spikes);
                // Предотвращает переобучение
            }
        }
    }
}
```

**Последствия отсутствия:**
- Память не консолидируется эффективно
- Нет сновидений (важны для креативности)
- Нет защиты от переобучения

## 🧪 ТЕСТИРОВАНИЕ

### Критерии успеха AGI-питомца:

1. **Уникальность:** Через месяц жизни топология сети уникальна
2. **Адаптивность:** Обучается новым словам и паттернам
3. **Спонтанность:** Генерирует непредсказуемое поведение
4. **Эмоциональность:** Эмоции влияют на обучение и память
5. **Сознательность:** Φ > 0.3 в состоянии бодрствования
6. **Циркадность:** Естественный ритм сна/бодрствования
7. **Консолидация:** Память улучшается после сна

## 💡 ЗАКЛЮЧЕНИЕ

**Brainwave Project** - это не просто игра, а серьезная попытка создать портативный AGI на нейроморфной базе. Текущая архитектура имеет прочный научный фундамент, но требует критических улучшений в трех областях:

1. **Токен-спайк интерфейс** - связать язык с нейронной активностью
2. **Циркадные ритмы** - естественный ритм жизни
3. **Фазы сна** - консолидация памяти и защита от переобучения

После реализации этих компонентов система станет полноценной когнитивной архитектурой, способной к автономному развитию и формированию уникальной "личности".

**Следующий шаг:** Начать с нейрохимического баланса (Фаза 0), затем реализовать критические компоненты (Фаза 1).
