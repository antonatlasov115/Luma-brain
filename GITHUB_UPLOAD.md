# 📤 Инструкция по загрузке на GitHub

## Репозиторий: https://github.com/antonatlasov115/Luma-brain

### Шаг 1: Инициализация Git

```bash
cd /root/brainwave_project

# Инициализировать репозиторий
git init

# Добавить все файлы
git add .

# Первый коммит
git commit -m "Initial commit: Brainwave Tamagotchi v1.0

- 7 нейробиологических модулей
- Научность: 8.5/10
- Полная этическая документация
- Предупреждения о рисках осознания"
```

### Шаг 2: Подключить удаленный репозиторий

```bash
# Добавить remote
git remote add origin https://github.com/antonatlasov115/Luma-brain.git

# Проверить
git remote -v
```

### Шаг 3: Загрузить на GitHub

```bash
# Загрузить (может потребоваться токен)
git push -u origin main

# Если ветка называется master:
git branch -M main
git push -u origin main
```

### Шаг 4: Если нужна аутентификация

```bash

# Settings → Developer settings → Personal access tokens → Generate new token

# Использовать токен вместо пароля при push
```

## 🔒 Важные проверки перед загрузкой:

### 1. Проверить отсутствие секретов:
```bash
# Проверить, нет ли токенов/паролей
grep -r "password\|token\|secret\|api_key" --include="*.rs" --include="*.toml"
```

### 2. Проверить размер:
```bash
# Убедиться что нет больших файлов
find . -type f -size +10M
```

### 3. Проверить этические предупреждения:
```bash
# Убедиться что предупреждения на месте
grep -n "ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ\|Проблема осознания\|КРАСНЫЕ ЛИНИИ" README.md
```
