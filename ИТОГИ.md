## 🚀 Как запустить:

### Текущая версия (v1):
```bash
cd /root/brainwave_project
rm -f pet_brain.json
cargo run --release -p brainwave_tamagotchi
```

### С улучшенной химией (v2):
1. Заменить в `brainwave_tamagotchi/Cargo.toml`:
```toml
chemistry_module_v2 = { path = "../chemistry_module_v2" }
```

2. В `main.rs` изменить импорт:
```rust
use chemistry_module_v2::{ChemistryModule, Neurotransmitter};
```

3. Пересобрать:
```bash
cargo build --release -p brainwave_tamagotchi
```
