# 🔐 Инструкция по загрузке (требуется аутентификация)

## Проблема: 
Git не может запросить логин/пароль в этом окружении.

## Решение 1: Использовать Personal Access Token (рекомендуется)

### Шаг 1: Создать токен на GitHub
1. Перейти: https://github.com/settings/tokens
2. Нажать "Generate new token" → "Generate new token (classic)"
3. Выбрать права:
   - ✅ `repo` (полный доступ к репозиториям)
4. Скопировать токен (показывается один раз!)

### Шаг 2: Загрузить с токеном
```bash
cd /root/brainwave_project

# Вариант A: Использовать токен в URL
git remote set-url origin https://YOUR_TOKEN@github.com/antonatlasov115/Luma-brain.git
git push -u origin main

# Вариант B: Ввести токен как пароль
git push -u origin main
# Username: antonatlasov115
# Password: [вставить токен]
```

## Решение 2: Использовать SSH ключ

### Шаг 1: Создать SSH ключ
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# Нажать Enter 3 раза (без пароля)

# Показать публичный ключ
cat ~/.ssh/id_ed25519.pub
```

### Шаг 2: Добавить ключ на GitHub
1. Скопировать вывод команды выше
2. Перейти: https://github.com/settings/keys
3. Нажать "New SSH key"
4. Вставить ключ

### Шаг 3: Изменить remote на SSH
```bash
git remote set-url origin git@github.com:antonatlasov115/Luma-brain.git
git push -u origin main
```

## Решение 3: Скачать архив и загрузить через веб-интерфейс

### Шаг 1: Создать архив
```bash
cd /root
tar -czf brainwave-tamagotchi.tar.gz brainwave_project/
```

### Шаг 2: Скачать файл
- Скачать `brainwave-tamagotchi.tar.gz` на локальный компьютер

### Шаг 3: Загрузить на GitHub
1. Перейти: https://github.com/antonatlasov115/Luma-brain
2. Распаковать архив локально
3. Загрузить файлы через веб-интерфейс или git на локальной машине

## 🎯 Рекомендация:

**Используйте Решение 1 (Personal Access Token)** - самый простой и безопасный способ.

Токен можно сохранить в git credential helper:
```bash
git config --global credential.helper store
# При первом push введите токен - он сохранится
```
