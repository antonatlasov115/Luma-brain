# 🧪 Улучшенная химия мозга v2.0

## Что нового:

### ✅ Реализовано (Уровень 1):

1. **Обратный захват (Reuptake)** - транспортеры очищают синапс
   - Разные скорости для разных медиаторов
   - Ацетилхолин убирается очень быстро (0.8)
   - Дофамин медленнее (0.3)

2. **Ферментативная деградация**
   - MAO разрушает дофамин, серотонин, норэпинефрин
   - COMT дополнительно разрушает дофамин
   - AChE очень быстро разрушает ацетилхолин

3. **Взаимодействия медиаторов**
   - Дофамин ↔ Серотонин (антагонизм)
   - Норэпинефрин → Дофамин (синергия)
   - ГАМК ↔ Глутамат (баланс возбуждения/торможения)

4. **Десенситизация рецепторов**
   - Хронически высокий уровень → снижение чувствительности (30% минимум)
   - Хронически низкий уровень → повышение чувствительности (150% максимум)
   - Постепенная нормализация к 100%

5. **Гомеостатическая регуляция**
   - Медленное возвращение к базовым уровням
   - Поддержание стабильности системы

6. **Эффективные уровни**
   - Учет чувствительности рецепторов
   - `effective_level = level × sensitivity`

## Использование:

```rust
use chemistry_module_v2::ChemistryModule;

let mut chemistry = ChemistryModule::new();

// Обновление (теперь с взаимодействиями!)
chemistry.update(hunger, energy, happiness, stimulation, elapsed);

// Выброс дофамина
chemistry.release(Neurotransmitter::Dopamine, 0.3);

// Эффективный уровень (с учетом рецепторов)
let effective_dopamine = chemistry.get_effective_level("Дофамин");

// Расширенный отчет
let report = chemistry.get_detailed_report();
for (name, level, sensitivity) in report {
    println!("{}: {:.0}% (рецепторы: {:.0}%)", 
             name, level * 100.0, sensitivity * 100.0);
}
```

## Пример динамики:

```
Секунда 0: Дофамин = 50%
Выброс +30% → Дофамин = 80%

Секунда 1:
- Обратный захват: 80% × exp(-0.3×0.1) = 77%
- MAO деградация: 77% × (1 - 0.15×0.05) = 76%
- COMT деградация: 76% × (1 - 0.1×0.03) = 75.8%
→ Дофамин = 75.8%

Секунда 2-10: (хронически высокий)
- Рецепторы десенситизируются: 100% → 95% → 90% → ...
- Эффективный уровень = 75% × 0.9 = 67.5%
→ Толерантность!

Секунда 20: (без новых выбросов)
- Уровень падает: 75% → 60% → 50% → 40%
- Рецепторы восстанавливаются: 90% → 95% → 100%
→ Нормализация
```

## Интеграция в тамагочи:

Замените `chemistry_module` на `chemistry_module_v2` в `Cargo.toml`:

```toml
[dependencies]
chemistry_module_v2 = { path = "../chemistry_module_v2" }
```

Код остается совместимым! Все методы те же, но теперь с реалистичной динамикой.
